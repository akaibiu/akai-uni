# Akai-uni

#### 介绍
uniapp小程序开发模板(集成Tuniao-ui、uView-ui、Z-paging)等高性能UI库和刷新加载插件。集成了request请求和upload上传等公共库，包含axios三层封装以及公共库封装。开箱即用!

#### 软件架构
uView-Ui Tuniao-Ui Z-paging Animate.css
